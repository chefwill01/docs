<Step title="Install ShadCN Component">
    Add the PatchProvider component to your project root.
    <CodeGroup>
      ```bash npm
      npx shadcn@latest add https://patch.bot/r/PatchWidget.json
      ```
      ```bash pnpm
      pnpm dlx shadcn@latest add https://patch.bot/r/PatchWidget.json
      ```
      ```bash yarn
      yarn shadcn@latest add https://patch.bot/r/PatchWidget.json
      ```
      ```bash bun
      bunx --bun shadcn@latest add https://patch.bot/r/PatchWidget.json
      ```
    </CodeGroup>
  </Step>
